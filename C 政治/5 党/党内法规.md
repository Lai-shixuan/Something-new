# 现在框架

其中现在的党内法规框架的原则是是，规范主体及行为，规范监督。具体文章分成5个部分进行讲述

1.  党章
2.  组织
3.  领导法规
4.  自身建设
5.  监督保障

党章：对党的性质和宗旨、路线和纲领、指导思想和奋斗目标、组织原则和组织机构、党员义务和权利以及党的纪律等作出根本规定，全面阐明党的政治立场、政治目标、政治路线、政治方针

党的组织法规：规定了党的组织结构、选举等环节，以及各组织职能

党的领导法规：规范了党与政府的关系，具体从五位一体战略布局去描述，并且最后增加党与国防军队环节的关系。

党的建设法规：按照新时代的政治、思想、组织、作风、纪律建设5个方面展开论述

党的监督法规：从监督、奖惩、保障3个方面进行论述
