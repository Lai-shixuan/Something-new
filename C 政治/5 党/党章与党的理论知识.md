# 历代领导集体的主要贡献

邓小平提出了中国特色社会主义道路；江泽民提出党是什么，党如何管党；胡锦涛指出如何发展；习近平将以上三个方面都继续发展了，因此称之为新时代，我在大纲中简称为“新社”。

# 党的方针

党的方针要分解成2个部分，一部分仅仅是党的建设的方针，而另一部分是党领导的国家的方针。

在叙述完历史的第一部分是**基本国情**的描述。

基本国情后，指出了党的基本路线，这既是党的，也是国家的，但更多是国家的。这是国家发展的最基本的方向。与之相对应的是党的组织路线、群众路线、理论路线，暗藏在党的建设的5个要点内。以上这4条路线，1+3均为党和国家的**前进方法**。

由于是党章，对于国家的方针政策写的并不全面。十九大报告中列出了十多项要求，或许是国家前进方法的有力补充。而对于党的前进方法，党章末尾通过5个要求予以明确。

党章在国家发展的基本方向后，写了国家发展的**目标**，即“五位一体+”。此处的加号，意思是军事、外交、爱国战线等。

国家的目标后是党的建设**目标**，分为七个部分。政治、思想、组织、纪律、制度、作风、反腐败斗争。

党的建设目标后，提到了上文所说的党的前进方法的5个要求。这也就到了党章的末尾。

党章的最后一段指出了党对于国家的**领导关系**。

但是，除此之外还想强调的是，在党章写五位一体的部分，同时附带的指出了4个全面布局。如下图所示，4个全面其实是大箭头的精简版箭头。全面建设社会主义现代化国家是党和国家的目标，深化改革是党和国家的发展动力，而依法治国和从严治党则是前进方法，是目前的主要矛盾[^2]。
