# 第一章 反对外国侵略的斗争

+ 第一次鸦片战争，《南京条约》后，有中英虎门条约，中美望厦条约，中法黄埔条约
+ 天津条约、北京条约是第二次鸦片战争先后签下的
+ 瑷珲条约是天津条约的俄国版
+ 北京条约分别和英、法、俄三个国家签订了条约，其中主要是同意了天津条约、瑷珲条约
+ 涉及香港岛的条约有，南京条约、北京条约、展拓香港界址专条，又名第二北京条约。
+ 中国半殖民半封建社会的原因，因为不是完全的资本主义社会，且不是完全的封建社会，因此是半殖民、半封建
+ 中国工人阶级最早出现于19世纪40至50年代外国资本主义在华企业中。因此，它是先于中国的资产阶级而产生的。
+ 当时中国社会出现的是工人阶级和资产阶级，后者可以分为官僚买办资产阶级和民族资产阶级
+ 1841年，广州三元里人民是第一批抗英的人民
+ 林则徐是近代中国睁眼看世界的第一人，组织翻译了英国人慕瑞的《地理大全》，编成《四洲志》
+ 魏源在1843年1月编成《海国图志》，并提出师夷长技以制夷而作。是在四洲志的基础上的 
+ 郑观应写《盛世危言》中，大力反正民族工商业，进行商战，设立议院，实现“军民共和”制度，1895
+ **1895年，签订了马关条约**
+ 孙中山喊出了“振兴中华”的时代最强音（在甲午战争之后）
+ 孙中山喊出，方今强邻环列，虎视鹰瞵，久垂涎于中华五金之富，物产之饶。蚕食鲸吞，已效尤于接踵；瓜分豆剖，实堪虑于目前。有心人不禁大声疾呼，亟拯斯民于水火，切扶大厦之将倾。
+ 严复喊出了“救亡图存”的口号，1895年，甲午战争后，并出版天演论。
+ 1898年，时局图被人绘制。
+ **1900年，八国联军侵华战争**
+ 辛丑条约签订于八国联军侵华战争，此时中国完全沦为半殖民地半封建社会
+ 庚子赔款是赔款最多的，1900年是庚子年
+ 庚子赔款最早由美国带头退还，因留美教育原因，后部分国家因一战胜利取消，苏联因反对帝国主义侵略取消，其他国家陆续取消。但是仍然给付到1938年

# 第二章

+ 《天朝田亩制度》是太平天国第一步革命纲领，提出了平均分配土地的方案。
+ 天朝田亩制度，具备先进性，也有其空想性
+ 近代第一个比较系统的发展资本主义的方案是，《资政新篇》
+ 洪仁干提出了资政新篇，但是并没有实施。即倡导资本主义的雇佣制度，从学习生产力变为学习生产关系
+ 洋务派，中体西用是指导思想，自强求富是目标
+ 洋务运动的首要目的是加强海防、边防（军事目的出发）
+ 洋务派发展了官督商办的一些企业，成为资本主义性质的企业
+ 洋务运动开始于1860年代，第二次鸦片战争，在恭亲王的组织下兴起
+ 洋务运动的失败
	1. 封建性，生产力与生产关系不相容
	2. 对外国具有依赖性，外国不希望真正富强起来
	3. 企业管理具有腐朽性，是封建衙门式的
+ 维新派与守旧派的论战，主要是
	1. 是否变法
	2. 是否兴民权、设议院、君主立宪
	3. 是否废八股、改科举、兴西学（但是没有否定中学，而是“托古改制”）
+ 维新派的缺陷在
	+ 不敢否定封建主义（包括封建的土地所有制关系）
	+ 对帝国主义报有幻想
	+ 惧怕人民群众

# 第三章

+ 孙中山成立的政党有，兴中会、同盟会、中华革命党、国民党
+ 同盟会的组成有，广东兴中会、浙江光复会、湖南华兴会、青帮、洪门等
+ 国民党由中华革命党改组而来
+ 清廷放弃抵抗外国侵略者的念头，甘当洋人的朝廷是在《辛丑条约》签订后
+ 1900年后，日本战胜俄国，获取东北利益；英国占据西藏；德国进入长江流域
+ 清末，资产阶级开始成为主要的角色，人民群众已经不能照旧生活下去了
+ 保路运动在四川激化，并最终导致辛亥革命
+ 时间线：辛亥革命-袁世凯就任大总统-二次革命-称帝-护国运动-府院之争-张勋复辟-护法运动-广东国民政府
+ 中国法律史：钦定宪法大纲——中华民国临时约法——中华民国约法——训政纲领
+ 询证纲领，国民党1928年指定的，宣布由军政时期进入训政时期
+ 三民主义，民生-民权-民族
	+ 民族：驱除鞑虏，恢复中华。放松了对帝国主义、封建残余势力的反对
	+ 民权：建立资产阶级民主共和国
	+ 民生：平均地价，不分所有权
+ 南京临时政府，希望通过承认清政府与列强签订的一切不平等条约和清政府所欠的一切外债，来换取列强支持。是它的局限性
+ 资产阶级革命派的根源失败原因是，民族资产阶级的软弱性和妥协性。换一个表述方式，在半殖民地半封建的中国，资本主义的建国方案行不通
+ 主观失败原因和根本原因有区别
+ 标志着旧民主主义革命的失败的是护法运动的失败
+ 辛亥革命反帝反封建不彻底，没有提出彻底的革命纲领
+ 辛亥革命没有改变中国人民悲惨的命运
+ 辛亥革命没有走自上而下的改革道路
+ 中华民国临时约法规定了行政、司法、立法的划分
+ 革命派反对了贫苦农民夺富人之田为己有的主张，这是他们的弱点
+ 革命派和改良的论战
	+ 是否进行革命手段推翻清王朝
	+ 是否推翻帝制实现共和
	+ 是否进行社会革命（平均地权）
+ 辛亥革命失败的原因是：
	+ 没有提出彻底的反帝反封建的纲领
	+ 不能充分发动和依靠人民群众，反封建的主力军——农民没有被动员起来
	+ 不能建立坚强的革命政党，没有强有力的核心

# 第四章

+ 新文化运动是1915年9月，陈独秀创办新青年开始的
+ 新民主主义是54之后，马克思主义才逐渐起到了主导作用，才是新民主的
+ 新青年的时候，基本口号，德先生、赛先生的目的是在中国实现西方式的资产阶级国家整个目标
+ 五四运动表达了反帝反封建的彻底性，和辛亥革命对应起来
+ 中国人民对帝国主义的认识先是感性的阶段，在义和团阶段
+ 之后，中国人民对帝国主义的认识是理性的，在五四运动中
+ 中国人民的民族意识普遍觉醒，全面觉醒，空前，甲午战争，五四运动，抗日战争
+ 党在二大的时候才提出了反帝反封建的民主革命纲领，指出了明确的斗争目标。1922年
+ 毛泽东称赞孙中山能够“适应世界之潮流，合乎人民只需要”，是因为孙中山提出了新三民主义，以及联俄联共扶助农工三大革命政策
+ 新三民主义内容：
	+ 民族主义：突出了反帝的内容，国内各民族一律平等
	+ 民生主义：平均地权，节制资本，后面还有耕者有其田
	+ 民权主义：为一般平民所共有，不应为少数人所得而私
+ 国共合作的政治基础是新三民主义
+ 大革命的失败，主要指的是反帝反封建的革命任务没有完成
+ 中国持续时间最长的以此罢工，就是省港大罢工
+ 全国范围的大革命高潮，1）五卅运动，2）省港（香港和广州）工人大罢工，3）国民政府在广州建立，4）北伐战争开始
+ 五四运动反对了孔子，但是不反对中国的全部文化
+ 新文化运动对孔学的批判是为了发展资本主义民主
+ 54以前新文化运动的局限性
	+ 将改造国民性放在首要位置（宣传和呐喊是没有作用的）
	+ 思想方法上，没有用马克思主义批判方法，有些人看问题很片面
	+ 新文化运动批判孔学，是为了发展资本主义民主。因此，没有提供新的思想武器彻底改造中国
+ 54运动不是工人阶级领导的，更不是它发动的
+ 54运动时间：1919年54
+ 新经济政策是1921年才实行的
+ 马克思主义没有给中国提供具体的行动方案，只不过给了行动指南
+ 马克思主义中国化的问题，是第六届六中全会上毛泽东提出来的,1938
+ 早期马克思主义思想运动的特点是，开始提出知识分子应该同劳动人民相结合的思想，主张知识分子“向农村去”，“到民间去”
+ 中国共产党一开始就是区别于第二国际旧式社会主义改良党的新型工人阶级革命政党
+ 大革命和以前的五四运动、辛亥革命相比，并不存在革命的背景更加复杂，仅仅是革命的社会内涵更加深刻
+ 大革命的失败，其中一个原因是党尚处于幼年，没有经验，对中国社会和中国革命的问题缺乏深刻的认识，不善于将马列主义原理和中国的革命实践结合起来
+ 早期马克思主义队伍主要有三种类型：
	+ 五四以前的新文化运动的精神领袖，如李大钊、陈独秀，是先驱者和擎旗人；
	+ 五四运动的左翼骨干，如毛泽东、周恩来等，是其主体部分；
	+ 三是部分原中国同盟会会员、辛亥革命时期的活动家，如董必武、林伯渠、吴玉章。
+ 第二次国共合作特点是：广泛的民族性和复杂的阶级矛盾，且双方都有政权，有军队的合作
+ 表明李大钊已经成为中国的第一个马克思主义者的文章是《我的马克思主义观》
+ 五四运动以前的新文化运动局限性有：
	+ 将改造国民性置于优先地位（仅靠宣传和呐喊的改造，而并非革命实践）
	+ 是为了建立资本主义共和国，但是当时资本主义建国的方案行不通
	+ 思想方法上，很多人没有马克思主义的批判方法，看问题片面

# 第五章 中国革命新道路

+ 南昌起义时间：1927年8月1日
+ 南昌起义是中国共产党独立领导革命战争，创建人民军队和武装夺取政权的开端
+ 湘赣边界起义公开的打出了工农革命军的旗帜，是以建立农村革命根据地为目标
+ 八七会议，整顿改编自己的队伍，纠正过去严重的错误，而找着新的道路
+ 毛泽东开辟新道路、创造新理论的革命首创精神的著作是 《反对本本主义》，1930
+ 1927年，毛泽东完成三湾改编，建立了支部建在连上的制度。
+ 党对军队的绝对领导，发端于南昌起义，奠基于三湾改编，定型于古田会议
+ 孙中山制定军政、训政、宪政时期
