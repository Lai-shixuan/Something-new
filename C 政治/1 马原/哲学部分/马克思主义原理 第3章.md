# 实践及认识及其发展规律

+ 非理性因素对理性因素既起着推动、调控的作用，同时又受到理性因素的决定与制约。因此没有起到决定性因素。
+ 理性因素在认识中起到了主导作用。
+ 人的感觉的的选择性是人的认识能力的界限？错，人可以通过认识工具来放大或延长自己的感官
+ 直接经验和间接经验，不存在谁检验谁，谁更有作用的说法
+ 真理同谬误的区别是，我们的对客体的认识是否与客体的实际相符合
+ 理性认识来源于抽象逻辑？错误，理性思维也需要社会实践，才能从感性上升到理性
+ 在真理的标准上，坚持唯物主义，就是要坚持”实践是真理的唯一标准“
+ 在真理的标准上，坚持辩证法，就是要坚持真理标准的确定性与不确定性的统一。
	+ 真理的标准是确定的含义是，实践作为标准是确定的，要反对怀疑主义、相对主义
	+ 真理的标准是不确定的，含义是实践可能有误差，有失误，有未被认识的现象，我们要避免教条主义
	+ 真理是一元的，真理本身没有主体的差别，而价值是多元的，不同的主体有不同 的需要、不同的价值要求和价值选择
+ 真理是具体的，如果超出一定的条件、范围，就会变成缪误
+ 真理是客观实在？错误，真理具备客观性，真理还有客观的内容，但真理并不是客观存在。
+ 必然王国和自由王国是人类在客观世界面前所处的两种不同的**社会活动状态**
+ 一种行为是否能够算做实践，关键在于是否超出了纯粹的意识活动，是否改变了除了实践主体的意识状态意外之外的其他生存物的状态。
+ 实践活动具有自觉能动性，并且主要以目的性来表示，但是这并不能用来作为实践活动的判别条件
+ 在人的认识活动中，摹写、反映的特性与能动、创造的特性，二者是不可分割的。认识是摹写性与创造性的统一。
+ 客观真理被证明是错的，应该理解为，真理被发展了，而非被推翻了
+ 唯物主义和唯心主义都相信经验，区别是，主观唯心认为经验是主观产生的，客观唯心认为经验是某种隶属于第一性的，不依赖于物质存在的；唯物主义认为经验必须是经过实践过得才行
+ 旧唯物主义的缺陷，脱离开了实践、辩证法，认为是“照镜子”式的直观反映论（消极被动）
+ 直接经验与间接经验是相结合的，而不是转化的
+ 感性认识和理性认识中，都有非理性因素


+ 感性和理性认识是相互渗透、相互包含的。人的感性是渗透着理性的感性，而理性以感性认识为材料，且可以透过语言等感性的方式表达出来
+ 属于唯心主义的观点是，意识和思维统一，如观念与主体感觉的符合，思维与先验形式的符合
+ 实用主义者的观点，也是唯心主义的观点
+ 真理的内容是客观的，形式是主观的
+ 价值具有客观性，价值是一种客观存在，独立于人们对它的认识和评价。价值评价是一种具有客观性的实践活动
+ 特殊的认识过程可以和实践同步，如预测等。
+ 必须把人的全部实践作为真理的标准，实践能够检验一切认识
+ 认识的反复性，决定了客观事物的复杂性，判断：错误，因为这是唯心主义的
+ 价值的特性：客观性、主体性、社会历史性、多元性
+ 价值是一种关系，取决于主体的需要和客体的特性
+ 任何成功的实践都是真理尺度和价值尺度的统一，是合规律性和合目的性的统一
+ 真理和价值在实践中辩证统一的关系：价值尺度必须以真理为前提，另一方面，人类自身需要的内在尺度，推动者人们不断发生新的真理
+ 实践的主体和客体相互作用关系中，最根本的关系是 实践关系，而不是价值关系
