# 世界的物质性及发展规律

+ 哲学的基本问题
	+ 思维和存在的第一性问题，即何者为本原的问题；分为唯物和唯心
	+ 思维和存在的同一性问题，即思维能否正确认识存在的问题；分为可知论和不可知论
+ 辩证法和形而上学，并不等于唯物和唯心，他们是和两大派别相结合的
+ 哲学中物质的概念，和自然科学中物质的概念，是共性与个性的区别
+ 客观必然性既不能摆脱或绕开，也不能废除或排除。认识世界到改造世界，是必然走向自由的过程
+ 认识：理性思维和非理性思维应该是协同起作用的结果
+ 事物的本质，必须通过理性思维才能认识
+ 实践的性质：客观性（真实现实性）、自觉能动性、社会历史性
+ 物质对意识具有反作用（能动作用）
	+ 具有目的性和计划性[^3]
	+ 意识活动具有创造性
	+ 意识具有指导实践改造世界的作用
	+ 意识具有调控人的行为和生理活动的作用
+ 物质世界具备人们未认识的现象，正确，并不是无法认识
+ **时空观**
+ 相对静止是事物存在和发展的必要条件；是人们认识和利用事物的前提；是过去运动的结果和未来运动的出发点；是理解和衡量运动的尺度。
+ 时间是一维的，即一去不复返；空间是三维的
+ 时间是物质运动持续性、顺序性的体现；空间是物质运动广延性、伸张性的体现
+ 时间和空间是物质运动的存在形式，是不能离开物质的，是客观存在的
+ 具有具体物质形态的时空是有限的，而整个世界是无限的
+ 绝对的与无条件的并列，相对的和有条件的并列
+ 时间和空间的客观实在性是绝对的，但具体特性是相对的。意思是客观实在性是绝对的，但是具体特性随着物质本身的性质，或者物质的运动的变化而变化。
+ 物质生产活动及生产方式是人类社会赖以存在和发展的基础
+ 物质生产实践是人类最基本的实践活动
+ 理性因素和非理性因素都可以认识物体的外在特性
+ 抽象（非现实）可能性是指，有根据，不充分，所以现在做不到但是未来可能做到现实可能性是指，有根据，比较充分，所以有可能性
+ 可能性和不可能性是，有没有现实根据。如水中捞月。
+ 事物发展的过程，从形式上看，是事物在时间上的持续性和空间上的广延性的交替；从内容上看，是事物形态、形式、结构、功能和关系上的更新。
+ 现象与本质是一对范畴，本质通过现象表示，现象中一方面的反映了本质。
+ 现象包含真相和假象，一个是正面的反映本质，一个是错误的歪曲的反映本质
+ 错觉和假象的关系，错觉不反映本质。假象是客观的，错觉是主观的
	+ 矛盾的同一性和斗争性是否是有条件的？矛盾的同一性是有条件的、相对的，矛盾的斗争性是无条件的、绝对的。
	+ 矛盾如何推动着事物的发展变化？矛盾的同一性和斗争性相结合，构成了事物的矛盾运动，推动着事物的变化发展。
	+ 哪些矛盾可以推动事物发展变化？事物的发展实质上就是新事物战胜旧事物，只有新事物反对旧事物的斗争，才能推动事物的发展。有时候是同一性占主导，有时候是斗争性占主导。
	+ 真理的内容是客观的，真理的形式是主观的
	+ 矛盾的普遍性和特殊性辩证关系的原理是马克思主义的普遍真理同各国的具体实际相结合的哲学基础。
+ 事物保持自己的质的稳定性的范围，是度
+ 度的两端，是关节点/临界点，超出了度的范围，食物就形成了新的质量统一
+ 肯定和否定的含义：肯定是保持事物存在的因素，否定是促使事物灭亡的因素
+ 形而上学的否定与辩证的否定的区别，前者是外在的否定[^1]，是绝对的、不含有肯定的否定；但是辩证的否是从内部的否定，是对自我的扬弃[^2]
+ 马克思主义活的灵魂，具体问题具体分析
+ 客观辩证法：客观事物相互作用相互联系的方式，呈现出各种物质形态的辩证运动和发展规律，不以人的意志而转移
+ 主观辩证法：是人类认识和思维运动的辩证法，以概念为思维细胞。可以称之为概念辩证法
+ 主观辩证法是客观辩证法在人的思维中的反映，本质上是统一的，但表现的形式不同
+ 两点论和重点论的含义，前者是事物的优点和缺点都存在，后者是矛盾的两面应该是有重点的，起主导作用的。事物的性质主要由取得支配地位的主要矛盾的主要方面所决定。
	+ 人对事物的认识经历着从（感性）具体到抽象和从抽象到（思维）具体的过程（具体一抽象一具体）
+ 人对事物的认识，起点是感性具体，终点是思维具体，中介是各种形式的逻辑中介
+ 分析是综合的基础，综合是分析的完成
+ 分析与综合比归纳和演绎更加深刻
+ 整体与部分的关系，是分析与综合的原型；一般与个别的关系，是归纳与演绎的原型。思维方法是有客观原型的
+ 逻辑与历史是统一的，原因是马克思毕生的科学活动（研究历史时），或是逻辑的方法，或是（历史）实证的方法，却从没有将两者对立起来，而是从揭示人类历史矛盾运动的科学性、真理性这一总体目的出发，将两者有机地结合在一起。
+ 逻辑与历史相统一是从抽象上升到具体的内在要求，历史是逻辑的基础和内容，逻辑是历史在理论上的再现，是“修正过”的历史。
+ 主观唯心主义把人的感觉、观念作为世界的本原（关键词：心、观念、感觉等）；
+ 客观唯心主义把某种脱离物质、脱离任何个人的精神作为世界的本原（关键词：理、理念、绝对观念等）。
+ 形而上学，割裂了自然世界和社会的物质统一性，认为运动和时空是割裂的，让相对静止为绝对的（形而上学不变论）。
+ 个性与共性，局部与整体，是两个层次的概念，很少并列提到
+ 从个性中看到共性，从相对中找到绝对，从暂时中发现永恒，这是马克思主义物质观体现的唯物辩证法。
+ 意志是物质的存在形式，错，运动是物质的存在形式。意志是物质的产物。
+ 意志可以通过实践转化成物质
+ 意识的主观性，表现在
	+ 对客观事物近似真实的印象
	+ 客观对象的歪曲的反映
	+ 对虚幻的反映
+ 意识对物质具备相对独立性，但是不能通过“画饼不能充饥“来体现
+ 事物的性质取决于人们的观察角度，错误，取决于主要矛盾
+ 社会发展的根本动力是，社会基本矛盾，而不是科技创新
+ 事物的联系是事物所固有的，这是普遍性吗？不是客观性
+ 对立面统一的含义：矛盾
+ 同一性是事物存在合发展的前提，而不是矛盾性
+ 矛盾双方中，一方的发展以另一方的发展为条件
+ 矛盾的双方被调和，可以结束矛盾吗？不能，最多是融合。调和否认了斗争性
+ 共性与个性、矛盾的普遍性和特殊性、一般与个别可以视为同一个层次的概念
+ 共性是共同点吗？个性是共性事物的表象差异特征，共性是个性事物的内在统一性特征。共性不是共同点，共性是本质相同。
+ 区分质是认识量的前提，考察量是认识质的深化
+ 扬弃，保留是发展中的连续性，客服，就是发展中的非连续性。所以扬弃是连续性和非连续性统一
+ 扬弃，实现了旧事物到新事物的发展，但同时通过扬弃联系起来，所以是联系和发展的统一
+ 只有正确的认识可以反作用于实践？错，认识都可以。
+ 源和流的关系，意思是源头和从源头来的流水的关系
+ 共性寓于个性
+ 辩证思维方法与科学思维方法：侧重点不同；前者是后者的方法论前提，后者丰富和发展了前者。
+ 世界不是既成事物的集合体，而是过程的集合体
+ 联系的普遍性，说明
	+ 事物内部不同部分是有结构的
	+ 事物不会孤立的存在，不会没有联系
	+ 事物是相互联系的统一整体，有整体和局部的关系
+ 4种辩证思维方法
	+ 归纳与演绎
	+ 分析与综合
	+ 抽象与具体
	+ 逻辑与历史相统一


[^1]: 如对蚂蚁的外在的否定，踩死了一只蚂蚁
[^2]: 内部的否定意味着，鸡蛋从内部打破，是一个新的生命
