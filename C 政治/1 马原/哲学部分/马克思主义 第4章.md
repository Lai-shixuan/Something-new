# 人类社会及其发展规律

+ 生产方式是生产力和生产关系的总和
+ 物质资料的生产方式构成了人类 社会存在和发展的基础，集中体现着**人类社会的物质性**
+ 人人创造历史，判断题：错误，人们创造自己的历史。人们创造自己的历史不能和社会历史划等号。社会历史不是个人历史的简单叠加。每一个人都是历史的创造者，错
+ 历史人物的出现体现了必然性和偶然性的统一
+ 生产工具是区分社会经济时代的客观依据
+ 劳动是人的存在方式，也是社会存在和发展的基础
+ 生产方式是社会历史发展的决定力量
+ 人民群众是社会形态变革的决定力量
+ 生产力是衡量社会进步的根本尺度
+ 科学技术是先进生产力的集中体现和主要标志
+ 生产力包括，劳动资料、劳动对象、劳动者
+ 生产关系是一种客观的关系，不以人的意志为转移
+ 经济基础是社会的骨骼系统，上层建筑是社会的血肉系统
+ 阶级斗争是阶级社会中社会基本矛盾的体现，是阶级社会发展的直接动力
+ 全部社会关系 分为物质关系和思想关系两大类，其中物质关系是社会存在的一部分，思想关系不是
+ 物质关系是不依照人们的意志为转移的。思想关系是人们的意识形成的关系，是物质的反映
+ 人的本质是社会属性，而不是自然属性
+ 人的本质不是人固有的，即不是所有的个体的人身上抽象出来的共同性
+ 阶级首先是一个经济实体
+ 社会规律必须通过有目的、有意识的人的活动（即自觉力量）才能起作用并实现出来
+ 人不是抽象的而是现实 的，现实的人及其活动是社会历史存在和发展的前提
+ 社会形态，还是 社会的经济形态、政治形态和意识形态三者的历史的、具体的统一
+ 社会制度是社会形态的同义语
+ 社会形态，是关于社会运动的具体形态，发展阶段，和不同质态的范畴，是同生产力一定阶段相适应的经济基础与上层建筑的统一体
+ 语言不是社会意识形态，不是上层建筑
+ 上层建筑可以分为政治上层建筑和观念上层建筑，前者是主导
+ 非意识形态的社会意识，是无阶级性的，例如语言学、心理学、自然科学的
+ 社会意识形态是观念上层建筑，是有阶级性的
	+ 政治法律思想，是主导的社会意识形态
+ 国家是阶级矛盾不可调和的产物
+ 上层建筑，如果它为落后的经济基础服务时,就成为阻碍社会发展的消极力量。所以可以是积极的也可以是消极的
+ 经济基础的性质由占支配地位的生产关系性质决定
+ 生产力具有社会历史性？正确
+ 生产力中最活跃的因素是，劳动者，而不是科学技术
