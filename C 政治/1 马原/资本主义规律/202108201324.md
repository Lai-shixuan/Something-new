# 资本的循环周转

资本有三种循环，他们的性质是 #flashcard  
资本有三种循环，同时存在，时间上有连续性。  
+ 连续性的原因是整个循环自然地 3 个部分；
+ 并存的原因，举例而言，如果所有资本同时存在商品资本和货币资本，那么此时缺乏生产资本，将立刻停产停工，循环中断。

资本循环的三个部分 #flashcard  
整个循环依次分成三个部分：购买阶段、生产阶段和商品阶段  
+ 购买阶段：货币资本，原因是要买生产资本；
+ 生产阶段：生产资本，此时劳动力将产生增值；
+ 商品阶段：商品资本，此时的商品必须卖出，才能实现货币的回收，这是“惊险的跳跃”[[202108191901#^f0f4f9]][^1]
