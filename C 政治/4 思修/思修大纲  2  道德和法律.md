# 第五章 道德

1. 道德以善恶为评价标准，依靠社会舆论，传统习俗和内心信念来发挥作用。
2. 道德的起源
	1. 劳动是道德起源的首要前提。劳动创造了人和人类社会，是道德起源的第一个历史前提
	2. 社会关系是道德赖以产生的客观条件
	3. 人的自我意识是道德产生的客观条件
3. 道德的本质
	1. 道德是反映社会经济关系的特殊意识形态
	2. 道德是社会利益关系的特殊调节方式
	3. 道德是一种实践精神，本质上是知行合一的
4. 道德的功能
	1. 认识功能：反映社会关系特别是反应社会经济关系的功效与能力
	2. 规范作用：在正确的善恶观指导下，规范社会成员在社会公共领域、职业领域、家庭领域的行为，并规范个人品德的养成，引导并促进人们崇尚向善
	3. 道德的调节作用：道德通过评价等方式，指导和纠正人们的行为和实践活动，协调社会关系和人际关系的功效与能力。道德评价是道德调节的主要形式
5. 道德的作用：
	1. 道德为经济基础的形成、巩固和发展服务，是一种重要的精神力量
	2. 道德对其他社会意识形态的存在有着重大的影响
	3. 道德通过调整人们之间的关系维护社会秩序和稳定
	4. 道德是提高人的精神境界、促进人的自我完善、推动人的全面发展的内在动力
	5. 在阶级社会中，道德是调节阶级矛盾和对立阶级之间开展阶级斗争的重要工具
6. 道德的作用的性质：道德发挥作用的性质与社会发展的不同历史阶段相联系，由道德所反映的经济基础、代表的阶级利益所决定。只有反映先进生产力发展要求和进步阶级利益的道德，才会对社会的发展和人的素质的提高产生积极的推动作用，否则，就不利于甚至阻碍社会的发展和人的素质的提高。
7. 在阶级社会中，道德是调节阶级矛盾和对立阶级之间开展阶级斗争的重要工具
8. 推进新时代公民道德建设的出发点和落脚点是：培养担当民族复兴大任的时代新人
9. 我国社会主义道德的核心和原则是：以为人民服务为核心，以集体主义为原则
10. 集体主义最高层次的道德要求：无私奉献、一心为公
11. **5.3 遵守公民道德准则**
12. 准则的分类有，家庭道德、社会公德、职业功德、个人品德
13. 奉献社会是社会主义职业道德中最高层次的要求，是最高指向
14. 民生之本，最大的民生：就业
15. 家庭美德是否包括男女平等？
16. 家庭美德是否包括邻里互助？


# 第六章 法律

1. 决定法律本质、内容和发展方向的根本因素是生产方式
2. 国家对权利和义务，即社会利益 和负担进行的权威性分配是 制定环节
3. 国家司法机关及其公职人员依照法定职权和程序适用法律处理案件的专门活动，称为法律适用
4. 法律实施和实现的基本途径是守法
5. 国家强制力并不是保证法律实施的唯一力量，还有法律意识、道德观念、价值观念等
6. 中国特色社会主义法律体系是由宪法相关法等多个法律部门组成的有机统一整体。这个有机统一整体的主干 是 法律（不是宪法）
7. 建设中国特色社会主义法治体系的重点是  建设高效的法治实施体系，不是法律保障，监督，制定规范
8. 社会主义法治最根本的保证  党的领导
9. 人民民主专政是国体，人民代表大会制度是政体。国体决定政体，政体反映国体。
10. **6.3 中国特色法治体系**
11. 法治体系的主要内容有
	1. 法律规范体系
	2. 法治实施体系
	3. 法治监督体系
	4. 法治保障体系
	5. 党内法规体系
12. 法制体系的意义：
	1. 规范是前提
	2. 实施是重点
	3. 监督是保障
	4. 保障是依托
	5. 党内法规是本质要求
13. 平等是社会主义法治的基本要求
14. 新时代我国法治建设的指导方针是  科学立法、严格执法、公正司法、全民守法
15. 全面依法治国的基本格局是 科学立法、严格执法、公正司法、全民守法
16. 全面依法治国的总抓手是 建设中国特色社会主义法治体系
17. 如何评价党的领导和依法治国：党和法、党的领导和依法治国是高度统一的，社会主义法治必须坚持党的领导，党的领导必须依靠社会主义法治
18. **6.4 中国特色社会主义法治道路**
19. 法治思维与 人治思维的分水岭在于  最高的权威是法律还是个人
20. 法治 和 德治的区别：
	1. 法律规范和道德规范也都具有必须遵守的性质
	2. 实现方式和实施载体不同
	3. 约束作用的内在要求和表现形式不同
	4. 行为人违反两种规范以后承担的后果不相同
21. 德治不是治国理政的基本方式，是重要方式
22. 人身自由  人们一切行动和生活的前提条件
23. **6.5 培养法治思维**
24. 不可违抗性，法律必须遵守，违反法律要受到惩罚。任何人不论权力大小、职 位高低，只要有违法犯罪行为，就要依法追究和承担法律责任。
25. 普遍适用性，指是指法律在本国主权范围内对所有人具有普遍的约束力。
26. 公平正义主要包括权利公平、机会公平、规则公平和救济公平。
27. 权力公平包含 :一是权利主体平等，国家对每个权利主体“不偏袒” “非歧视”;二是 享有的权利特别是基本权利平等;三是权利保护和权利救济平等。
28. 正当程序的正当，表现为程序的合法性、中立性、参与性、公开性、时限性等方面（法律思维）
29. 行政保护是权力保障的关键环节（也是一种法治思维）
30. 尊重和维护法律权威的基本要求是，信仰法律、遵守法律、服从法律、维护法律
31. 分权制衡，不是宪法的基本原则
32. 交易自由，等价有偿不是民法的基本原则
33. 既是社会主义法治观念的核心要求，也是建设社会主义法治国家的前提条件  尊重和维护法律权威
34. 法律有无权威的一个因素（一共4个）：法律是否被人民信仰或尊敬
35. **6.6 依法行使权力和履行义务**
36. 公民参加国家政治、经济与社会生活的基础是 人身权力
37. 权力和义务是相互依存的，是目的与手段的关系，且可以具有复合型
38. 马克思认为，权利的产生、发展和实现，都必须以一定的社会经济条件为基础。
