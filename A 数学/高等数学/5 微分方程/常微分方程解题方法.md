# 一阶

一阶可分离变量的方程，分离变量到两边后进行积分。公式如下：
$$\int p(y)dy =\int q(x)dx +C$$

一阶齐次方程，注意$xy$也代表为2次，和$x^2$，$y^2$一样。齐次方程同时除以$x^2$，可以使$u=y/x$进行代换，然后转为可分离变量的形式。公式如下：
$$u=\frac{y}{x},\quad y'=u+xu'$$
$$\int r(u)du =\int q(x)dx +C$$

如果以上方式都不行，那么用一阶线性微分方程的公式去解题。顺便，以上两种类型也可以用该方法进行解题：
$$若\quad y'+p(x)y=q(x)$$
$$则\quad y=e^{-\int p(x)dx}[\int q(x)e^{\int p(x)dx}dx+C]$$

## 可降解的高阶（二阶）

首先，如果微分方程中一边只有高阶的倒数，另一边只有$x$，没有低阶的$y$，$y’$，那么可以使用如下公式，累次使用可以最终得到$y$：
$$y^{(n-1)}=\int p(x)dx+C$$

如果微分方程没有$y$，有$x$和$y'$，那么可以使用如下代换：
$$p=y',\quad y''=p'$$

如果微分方程没有$x$，有$y$和$y'$，那么可以使用如下代换（注意，此时$y''$的代换方式从p对x微分变成了p对y积分，目的均是为了方便移项积分）：
$$p=y',\quad y''=\frac{dp}{dx}=p\frac{dp}{dy}$$

## 二阶（高阶）

对于齐次方程，意味着没有x

$$y''+py'+qy=0$$
$$st.\quad r^2+pr+q=0$$

对于解，如果是不相等实数根的，$r_1$, $r_2$

$$y=C _ { 1 } e ^ { r _ { 1 } x}  +  C _ { 2 } e ^ { r _ { 2 } x}$$

如果是相等实数根:

$$y=(C_1+C_2x) e ^ { r_1  x }$$

如果有复数解，那么应该是成对出现的，则，

$$y = e ^ { a x } (C_1cos\beta x+C_2 sin\beta x)$$

对于非齐次方程，意味着是齐次方程的解的线性空间
