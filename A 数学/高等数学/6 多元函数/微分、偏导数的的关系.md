多元函数可微与连续的关系 #flashcard  
可微和连续的关系，可以类比连续和可导的关系。当出现尖点的时候，依旧连续，但是不可微  
<!--ID: 1639474300616-->



+ 一阶偏导数连续则可微，是通过拉格朗日定理证明的。[^1]

+ 一阶偏导数连续则可微，逆否命题：不可微则一阶偏导数不连续

+ 可微但是一阶偏导数不连续，可以举一个例子：$f ( x , y ) = x ^ { 2 } \sin ( \frac { 1 } { x } ) + y ^ { 2 } \sin ( \frac { 1 } { y } )$[^2]，图像如下
