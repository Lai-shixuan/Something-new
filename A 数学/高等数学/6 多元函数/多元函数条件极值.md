$$F _ { x } ^ { \prime } = f _ { x }' +\lambda\phi_x'+\beta\mu _ { x } ^ { \prime } = 0 $$

$$F _ { y } ^ { \prime } = f _ { y }' +\lambda\phi_y'+\beta\mu _ { y } ^ { \prime } = 0 $$

$$F _z^ { \prime } = f_z ^ { \prime } + \lambda \phi z'+\beta\mu _z^ { \prime } = 0 $$

$$F_\lambda'=\phi(x,y,z)=0$$
$$F_\beta'=\mu(x,y,z)=0$$

解出一个合适的(x,y,z)则比较大小，找到最大最小值

几何解释是，对F(x,y,z)画出等梯度线，然后，如果和约束相切，即为所求的点
