费马定理及其证明 #flashcard 


定理内容
$$if\quad f(x)在x_0处可导，且为极值点，$$
$$s.t.\quad f'(x)=0$$
证明：
假设现在是极大值，
$$f ^{ \prime } _{-}  ( x _ { 0 } ) = \lim\limits _ { x \rightarrow x _ { 0 }^- } \frac { {f ( x )} - {f ( x _ { 0 } )} } { x - x _ { 0 } } \geq 0 $$
$$f ^{ \prime }_ { + }  ( x _ { 0 } ) = \lim\limits _ { x \rightarrow x ^ { + } } \frac { {f ( x )} - {f ( x _ { 0 } )} } { x - x _ { 0 } } \leq 0$$
$$\therefore  f ^{ \prime }_ { + }  ( x _ { 0 } ) = f ^{ \prime } _{-}  ( x _ { 0 } ) = 0$$
