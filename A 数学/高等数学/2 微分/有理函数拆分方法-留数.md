有理函数拆分方法-留数 #flashcard  

分为无重根一元，有重根一元，无实数根二元：  
1. 无重根一元，如果是一元，带入除自己外的原函数；如果该元含有重根，此处计算的是二次项的分母  
2. 有重根一元，需要对除自己以外的原函数求导，之后带入（注意因式分解方法，此处计算的是一次项分母的系数）  
3. 无实数根二元，解出复数根后（正负号取正的），带入除自己以外的原函数  
  
例 7：[^1]  
$$\frac { 3 x + 6 } { ( x - 1 ) ^ { 2 } ( x ^ { 2 } + x + 1 ) } = \frac { A } { x - 1 } + \frac { B } { ( x - 1 ) ^ { 2 } } + \frac { D x + E } { x ^ { 2 } + x + 1 }$$
