**平面图形的面积**

$$S=\int_a^b |y_2(x)-y_1(x)|dx\tag{1}$$

$$S=\frac{1}{2}\int_\alpha^\beta |r_2^2(\theta)-r_1^2(\theta)|d\theta\tag{2}$$

**旋转体的体积**

绕x轴：
$$V=\pi \int_a^b y^2(x)dx\tag{3}$$
$$V = \pi \int_a^b |y _ { 1 } ^ { 2 } ( x ) - y _ { 2 } ^ { 2 } (x)|dx $$

绕y轴：
$$V=2\pi \int_a^bx|y(x)|dx\tag{4}$$
$$V = 2 \pi \int_a^b x | y _ { 1 } ( x ) - y _ { 2 } ( x ) | dx$$
