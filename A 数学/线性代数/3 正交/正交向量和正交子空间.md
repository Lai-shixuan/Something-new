# $A^TA$的特殊性质

这是一个非常重要的矩阵，在后面的证明中经常会用到，以下是一些基本的性质

1. 转置是其自身
2. $A^TA$将得到一个方阵
3. $N(A^TA)= N(A)$
4. $rank(A^TA) = rank(A)$
4. $A^TA$是可逆的，只要$A$是线性不相关的[^4]

**PS:** 线性不相关，代表可逆

$rank(A^TA)=?$ #flashcard  
$rank(A^TA)=rank(A)$，因为$N(A^TA)= N(A)$，线性空间相等意味着线性空间的维数相等
