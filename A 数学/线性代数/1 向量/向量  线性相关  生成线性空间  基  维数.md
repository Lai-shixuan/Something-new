# 线性相关

+ 向量组 are 线性相关、线性无关
+ 矩阵 has 秩
+ 向量空间 has 基

向量组是线性无关的，意味着方程$Ax=0$的Nullspace只有0向量（因为他们的线性组合不可能=0)。用矩阵的术语，n=r[^2]
向量组是线性相关的，意味着：方程$Ax=0$的Nullspace有一些非零的向量，用矩阵的术语，n>r（因为自由列的实质是，他们使得主列可以线性组合）

如果向量组中有一个0向量，那么这个向量组一定[^4]是线性相关的，因为一定有一种线性组合[^1]使他们和为0。

# 生成线性空间

对于某些向量空间，是由多个向量的线性组合生成的。我们将这些向量的线性组合视为一个subspace，那么这就是一个生成的线性空间

# 基 Basis

向量空间的一组基， 是一些重要的向量，具备两种性质 #flashcard 
1. 它们线性无关
2. 它们能生成整个空间
*换句话说，这些向量是一些重要的向量，因此剩余的不再重要*

那么，如果要描述一个向量空间，请用向量空间的基，因为这已经给出了所有与该向量空间相关的信息。

如果有几个向量，能否检测他们是否能生成一个线性空间？ #flashcard 
把他们组成一个矩阵，并判断 r 与 n 的关系。如果$r=n$，那么线性无关；如果$r<n$，那么线性相关

# 维数

维数的含义 #flashcard 
如果空间是一个线/面/体/……，那么它的基向量有1/2/3/……个
那么，上面这个个数，就是**维数**。
空间可以处于某种更大的范畴内，比如 $R^4$，此时最大允许的维数是4维
