# 方程的可解性

1. b是A的线性空间内
2. 如果A的某些行的线性组合得到0，那么相同行的b的线性组合也必须得到0

如果非齐次方程可解，求解的方法如下 #flashcard 
1. 获得增广矩阵$\begin{bmatrix}A&b\end{bmatrix}$
2. 设置所有的自由变量为0
3. 解方程，仅考虑主列和主行（此时可以求出一个特解来，且2个自由变量为0）[^1]
4. 加上通解的线性组合（Nullspace）

事实上这一通解和特解的加和，适用于线性空间的任何情况

从几何学来看非齐次线性方程的解 #flashcard 
+ 方程的解成为一个从原点移动去特解的线性空间[^2]
+ 这也解释了为什么，对自由变量赋值为0。因为这两个变量不参与线性空间的形成。

# 方程的解的特性总结

一个$m\times n$的矩阵A，拥有秩(r)，毫无疑问，$r<=m, r<=n$

假设，r=n（列满秩），那么没有自由变量，Nullspace(A) 只有0向量，因此$x_{particular}$成为唯一的解，因此要么0解，要么1解
此时，如果行不满秩，那么发现某些行是其他行的线性组合，那么对b有类似的要求，某些b是其他b的线性组合。

假设，r=m（行满秩），说明对于b不存在上述线性组合的要求，即对于b没有要求，方程均可解。
如果r=n=m，那么没有Nullspace，因此有唯一解。如果n>m，即n>r，存在自由变量，那么方程的全解是通解+特解
另外，矩阵可逆（因为是方阵、秩为r），其行最简形应该是单位阵

**矩阵的r告诉了我们所有的解的个数的信息**

当b=0向量时，对于解方程的意义： #flashcard 
那么此时必然出现一列没有主元，n>r，因此有无穷解，或者全0解

根据 $r$ 判断解的个数 #flashcard 
以下不包括全0解的情况。
+ $m>r$ 情况，可能有0解
+ $n>r$ 情况，可能有无穷解。
+ $n=r$ 情况下，可能有1解
