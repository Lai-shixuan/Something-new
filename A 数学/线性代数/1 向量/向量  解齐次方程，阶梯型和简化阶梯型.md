# 阶梯型解法

如果我们进行消元，只会改变系数矩阵A的线性空间，而不改变Null space，但改变了列空间。*WHY*

如何找矩阵的秩(Rank)？ #flashcard 
就是主元的数量
<!--ID: 1638588768860-->


齐次线性方程的解法： #flashcard 
1. 首先，矩阵可以化简成行阶梯型，可以看出分成了两种colums。
	1. 主列
	2. 自由列——对应的未知数可以任取，称为自由变量
2. 将自由变量任取，如1个自由变量取$[1]$，2个自由变量取为$[1,0]'，[0,1]'$ 
3. 可以回待解出主列对应的未知数。因此得到若干个特解。一般，有几个自由变量可以解出几个特解。
4. 进而，Ax=0的特解的线性组合将完全的覆盖得到整个Null space。换句话说，我们将矩阵得到了$Ax=0\rightarrow Ux=0$，并解出了新方程的解。该解恰好是旧方程的解。
   
*前提是有特解，即至少有一个自由变量；如果没有自由变量，Null space应当为0向量*
<!--ID: 1638589540966-->

对于一个$m\times n$矩阵，rank = r，自由变量和特解的数量是： #flashcard 
获得$n-r$个自由变量。因此我们得到$n-r$个特解（关于方程$Ax=0$的）
<!--ID: 1638589902389-->

若在阶梯型看到一行全为0、或者任何一列不是主元的，这意味着： #flashcard 
该行是主元行的线性组合，那么该列是主元列的线性组合。

# 简化阶梯型解法

通过简化阶梯型解齐次线性方程 #flashcard 
矩阵U进一步可以简化成矩阵R，即形成了简化阶梯型[^2]  [^3]
In which，可以见到一个单位阵，出现在主行和主列的交汇处。
Ax=0 -> Ux=0 -> Rx=0，解没有发生改变
因此如果我们讲，如果将矩阵化简成rref形式，可以观察到是$\begin{bmatrix}I & F\end{bmatrix}$[^4]，因此，它的Nullspace应当是$\begin{bmatrix}-F \\ I\end{bmatrix}$
*例子：此时F，I的列数互换，此前假如$rank=r$有$F=r\times 1$（自由列），$I=r\times r$ 或 $r\times (n-1)$，那么解空间中，$I=1$，剩余由$-F$来填空，即获得 $r+1=n$ 行*
<!--ID: 1638590728841-->


原因如下：
$\because\begin{bmatrix}I&F\end{bmatrix}\times\begin{bmatrix}x_{pivot}\\x_{free}\end{bmatrix}=0$
$\therefore X_{pivot}=-F\times X_{free}$，而$X_{free}=I$，即$X_{pivot}=-F$[^5]

一个重要结论，矩阵转置后主元数量： #flashcard 
不变
 
[^2]: 让主元上下均为0，且主元化为1
[^3]: 在MATLAB中，通过rref就可以一步得到简化阶梯型。
[^4]: I是由主元所在的行和列交汇得到的，而F是由自由列和主元所在的行交汇得到的
[^5]: 此处，X指代列向量组。另外由于任何向量和单位向量相乘，都是其本身
