# START

一切线性变换所需要知道的，就是对基的线性变换（假如有 n 个）

如果以特征向量为基，那么矩阵将会得到 $\Lambda$

课程最重要的是告诉我们如何求出能表示线性变换的矩阵，其就相当于待定系数法：  
我们给定的矩阵是在基 1 下的 INPUT 矩阵，  
我们要的矩阵是在基 2 下的 OUTPUT 矩阵  
$A\times INPUT=OUTPUT$
