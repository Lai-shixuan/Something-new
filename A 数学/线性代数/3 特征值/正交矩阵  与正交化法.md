# 标准正交基的好处：

如果把它们放入一个矩阵中：

$$Q^TQ=I$$

证明可以用到上面的性质

如果他是方针，我们给他一个特定的名称叫做正交阵，且Q是可逆的，意味着
$$Q^T =Q^{-1}$$

否则，我们把正交的矩阵称为标准正交矩阵

PS：转置矩阵，换行放在A左侧，换列放在A右侧

PS：一个世界超喜欢的方阵：Adhemar matrix

![[Pasted image 20210909224007.png]]


PS: 施密特正交化很讨厌，有根号出现

**好处**

转置矩阵P可以化简称为QQ^T，并且对它验证两个性质都可以成功

此后，如果将Q当成一个正交方阵，那么也可以验证方阵的性质，QQ^T = I（几何意义上来说，如果是一个正交方阵，那么充斥整个空间，于是投影矩阵应改成为I，即不用投影）

本章节一切复杂方程，在Q下都变得简单了。（如果Q是正交方阵，应该更简单）

如，A^TAx^=A^Tb，那么会化简称为X^=Q^Tb。进而，意味着往线性空间做投影是这个样子的，且往一个向量做投影变成了X = q_i^t b（带证明）

PS：验证是否线性无关的方法，如果m<n，那么必然是线性相关的。EX：在一个二维的空间有三个向量。
如果m>n，且A线性无关，那么必然这个C(A)没有充斥整个R^m空间
如果m=n，且A线性无关，那么C(A)必然充斥整个R^m空间
