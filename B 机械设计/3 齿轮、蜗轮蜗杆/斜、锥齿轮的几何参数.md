# 斜齿轮标准、直径、当量齿数 #flashcard 
1. 法面模数是标准值
2. 斜齿轮的分度圆直径：$d=m_n/cos\beta\times z$，$\therefore \beta$ 变大，$d$ 变大[^1]
3. 斜齿轮近似方法：与法面齿形相当的直齿圆柱齿轮
4. 斜齿轮当量齿数：$z_v=z/cos^3\beta$
