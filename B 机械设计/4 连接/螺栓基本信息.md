# 螺纹的基本参数和术语

+ 大径 d：公称直径，用于测量、描述，如M18中的18
+ 小径 d1：危险截面，用于校核强度[^2]
+ 中经 d2：用于几何计算直径
+ 线数，螺旋线的数量[^3]


# 螺纹的分类

+ 按照性状分类；三角形、矩形、梯形、锯齿形、圆弧形
	+ 三角形螺纹小心应力集中，所以去头
	+ 矩形螺纹曾经用于传动[^4]，后因为应力集中，所以用了梯形传动；
	+ 三角螺纹：强度高
	+ 锯齿形螺纹：一边传动，另一边强度高。
	+ 圆弧形螺纹：用于玻璃器皿。
+ 左旋螺纹、右旋螺纹
	+ 右旋、左旋判断方法：把轴立起来，左（右）测高就是左（右）旋
	+ 特点：瞬时间旋合，逆时针旋松
+ 内螺纹、外螺纹
+ 柱螺纹、锥螺纹、管螺纹

区别，由于角度更小，所以密封性更好一些

+ 粗牙、细牙
	+ 粗牙的螺距更高，因此强度更高；细牙用于，如笔筒等，经济性好，一般用于管螺纹
	+ 粗牙用的比较多，加工方便
	+ 细牙不耐磨，结构上小
