# 螺纹的基本参数和术语

+ 大径 d：公称直径，用于测量、描述，如M18中的18
+ 小径 d1：危险截面，用于校核强度[^2]
+ 中经 d2：用于几何计算直径
+ 线数，螺旋线的数量[^3]


# 螺纹的分类

按照性状分类；三角形、矩形、梯形、锯齿形、圆弧形

6种常见螺纹，多了一个圆弧形螺纹 #flashcard  
三角形螺纹小心应力集中，所以去头  
矩形螺纹曾经用于传动[^4]，后因为应力集中，所以用了梯形传动，现在即将被替代；  
梯形螺纹：强度高  
锯齿形螺纹：结合了传动以及强度，因此只能单向传递  
圆弧形螺纹：用于玻璃器皿  
管螺纹：紧密型比较好适用于管接头等依靠螺纹密封的附件

+ 左旋螺纹、右旋螺纹
	+ 右旋、左旋判断方法：把轴立起来，左（右）测高就是左（右）旋
	+ 特点：瞬时间旋合，逆时针旋松
+ 内螺纹、外螺纹
+ 柱螺纹、锥螺纹、管螺纹

区别，由于角度更小，所以密封性更好一些

粗牙、细牙螺纹的区别 #flashcard  
+ 粗牙：  
	+ 螺距更高，因此螺纹牙强度更高
	+ 用的比较多，加工方便  
+ 细牙：  
	+ 抗拉强度高  
	+ 螺距小，升角小，自锁性好  
	+ 结构上小，不耐磨，螺纹牙强度不如粗牙
