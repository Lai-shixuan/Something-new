- # 螺纹结构
	- 螺旋升角：$S=\pi\times d_2 tan\gamma$
	- 牙形角，$\alpha$
	- 牙侧角，$\beta$，牙有两个侧壁，因此有两个牙型角，$\beta_1-\beta_2=\alpha$
	- 工作高度 #先不学 
- # 自锁问题
	- 当量摩擦系数：当两个相对滑动的物体，处于非平面状态时候，通过当量摩擦系数简化为平面形式。具体来说，当两个物体是槽型相对滑动的时候，$f_Δ=f/sin(\theta/2)$，$\theta$ 是槽型夹角，此处$\theta=\alpha$[^1]
	- 当量摩擦角：假设有一个无穷大的推力沿着螺栓推出的方向推动的时候，其摩擦面分别受到外力造成的摩擦力和旋出方向滑动的力，若前者大于后者，即$F\times cos\beta\times f_v > F\times sin\beta$，那么永远无法滑出，实现自锁[^2]  [^3]。公式化简为：
		$$\varphi_v=arctanf_v=arctan(f/cos\beta)$$
	- 自锁条件：$\gamma<\varphi_v$
- # 传动效率：
	- 拧紧（正行程）：$$\eta = \frac{tan\gamma}{tan(\gamma+\varphi_v)}$$
	- 下降（反行程）[^4]：$$\eta = \frac{tan(\gamma-\varphi_v)}{tan\gamma}$$
	- 自锁螺纹传动效率比较低，通常低于50%
	- 梯形螺纹传动效率较高，$\beta$角较小
	- 双线梯形螺纹比单线梯形螺纹传动效率更高，因为$\gamma$越大，$\eta$越大
	- **传动效率和当量摩擦角（牙侧角）存在矛盾关系**
- # 常用螺纹类型
	- 普通螺纹（三角形螺纹）
		- 粗牙螺纹：相同公称直径下螺距大，强度高
		- 细牙螺纹：螺距小，自锁性好，适用于薄壁件，如管螺纹等，但是容易滑扣
			- 细牙螺纹自索性好的原因是：虽然$\beta$小，$\therefore\varphi_v$小，但是导程角$\gamma$更小，因此$\gamma$<$\varphi_v$。自锁性好
			- ![[Pasted image 20211003204034.png|600]]
			- 细牙螺纹抗拉强度高，但是螺纹牙强度不如粗牙螺纹
	- 矩形螺纹
		- 传动效率高，牙侧角约为0度，当量摩擦角约为0行。
		- 没有正压力的，因此没有丝毫的自锁性
	- 锯齿形螺纹
		- 一面三角形，一面矩形，结合2个螺纹的优点
		- 一般是用近乎矩形的面进行传动。其牙侧角$\beta$小，导致$\varphi_v$小，少摩擦，无法自锁
- # 连接方式
	- 普通螺栓连接：光孔，两侧都有操作空间。被连接件和螺栓之间有空隙
	- 加强杆螺栓连接（铰制孔螺栓连接）
		- 没有空隙，但加工精度高，定位精度高
		- *工作方式，是给两个截面之间设计预警力）*
		- *工作方式，会被螺栓杆进行承担，承受力更大*
		- 属于过渡配合
	- 螺钉连接：另外一边是盲孔
	- 双头螺柱：需要经常拆卸*可以看成是螺钉连接的改良*
	- 紧定螺钉链接：螺钉微端，顶住坑。但是传递较小的力。尖端或者平端都有。
	- 其他方式
		- 地脚螺栓连接
		- T型槽螺栓连接：铝型材连接用的
		- 拉环：
