- 结构
- 运动分析
	- 多边形结构
		- 分度圆直径计算
			- $$d=\frac{p}{sin\frac{180^\circ}{z}}$$
		- 平均速度
			- 平均速度=位移除以时间。位移等价于“转过一圈走过的链长”（位移不通过圆的周长来求）
			- $$v_m=\frac{n_1z_1p}{60\times 1000}=\frac{n_2z_2p}{60\times 1000}$$
		- 瞬时速度，通过半径乘以角速度求取
			- $$i = \frac { \omega _ { 1 } } { \omega_2 } = \frac { R _ { 2 }\cos \gamma } { R _ { 1 }  \cos \beta}$$
		- 产生了动载荷
			- 动载荷可以包括1）加速度可求，**2）** 剧烈变化的冲击载荷，加速度不可求，3）疲劳问题，应力周期变化。[^1]
			- 后果
				- 产生横向振动（上下方向）
				- 附加动载荷、惯性冲击载荷
- 受力分析
- 功率
	- $$P_0=\frac{P\times K_A}{K_zK_m}$$ ^357452
	- $K_z$是小链轮齿数系数
	- $K_m$是链排数
- 失效形式 ^c42309
	- 磨损：销轴和套筒铰链磨损（润滑良好和润滑不好形式），承载力较大时候
	- 销轴与套筒胶合，发生在高速油膜破坏或润滑不良的情况
	- 疲劳破坏：滚子和套筒、链板（以上3个组成了链传动的额定功率曲线）
	- 静强度：链条拉断（0.6 m/s)
- 布置：
	- 张紧轮应该在小带轮的松边上，外侧，以控制垂度
	- 张紧轮可以是有齿轮子，或者无齿的辊轮
	- 小带轮在上
