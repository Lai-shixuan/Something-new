# 综合

1. 效率，齿轮>链传动>带传动>蜗轮蜗杆
2. 承载能力，齿轮>蜗轮蜗杆>带传动=链传动，线速度同
3. 寿命，齿轮>带传动=链传动，*蜗轮蜗杆未知*
4. 噪音（仅带传动、蜗轮蜗杆无噪音）
5. 成本（齿轮、蜗轮比较昂贵）
6. 传动平稳性（仅链传动不好）
7. 传动同步比（仅带传动不好）

# 带传动

优点：
1. 挠性带平稳吸振，因此噪声小
2. 不用润滑油，噪声低（只接触面）
3. 过载保护
4. 承载能力较大，和链传动类似，不如齿轮传动

缺点：
1. 传动效率低，弹性滑动
2. 同步比不准确（同步带除外）
3. 传递同样大小圆周力时候，轴和轴承承受的压轴力比咬合传动大[^1]
4. 带的寿命较短[^2]

# 链传动

优点：
1. 效率较高，可以达到98%
2. 张紧力小，轴与轴承承受的力较小
3. 同步比比较准确
4. 低成本（和带传动相同

缺点：
1. 传动平稳性较差，有噪音
2. 寿命较低，没有齿轮高
3. 速度不均匀，有冲击和振动

# 齿轮传动

优点：
1. 速比精确、传动平稳
2. 承载能力大，功率：最大可达 $1\times 10^5 kw$，速度：最大可达 $300 m/s$
3. 效率高98~99.5%、
4. 寿命长
5. 结构紧凑
6. 尺寸：D——33m。
7. 适用场合：广泛

缺点：
1. 需要专门的加工设备；
2. 轴间距较大时需多级传动
3. 有噪音

# 蜗轮蜗杆

优点：
1. 高传动比
2. 无噪音
3. 工作平稳无冲击振动

缺点：
1. 成本较高
2. 效率低，发热大
3. 不宜大功率、长时间工作
